Class: Data

Data(envAdapter, files, options)

Data class ---------- Represents a collection of files (images, video, audio, text) with support for tensor conversion, result storage, and derived artifacts. Usage: const data = new Data(envAdapter, [file1, file2]); await data.load(); const tensors = await data.create_tensors("resnet50");
